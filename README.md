# Singlecell_oligodendrocytes

This repository contains data and analysis scripts for single-cell 
RNA sequencing of oligodendrocytes.

## Folder Structure
- Codes/ — R/Python scripts for analysis
- Figures/ — Main manuscript figures
- Supplemtary_Figures/ — Supplementary figures
- Tables/ — Result tables

## Requirements
- R version 4.x
- Seurat package
